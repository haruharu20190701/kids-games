# こどもあそびば — プロジェクトメモ

## ねらい
子供（5歳前後）向けのゲーム・教育コンテンツを**少しずつ増やしていくコレクション**。
「軽い・安全・どの端末でも遊べる・できた！が気持ちいい」を大事にする。

## 現状（2026-06-21 時点）
- 公開中: https://haruharu20190701.github.io/kids-games/
- ゲーム: **ぱずるぶろっく**（1本）
  - 8×8盤、**ステージ制**（決まった手数内に目標スコアでクリア）
  - 初期盤面に**ランダム配置**、手持ちは1つ置くごとに**個別補充**
  - **連鎖コンボ＋一気消し演出**（画面シェイク・紙吹雪・スコアポップ・音階上昇）
  - **猫の相棒**（設置時セリフ100種・吹き出しは猫の右）
  - **スマホ/タブレットの縦横**に対応（回転で状態保持リレイアウト）
- メニュー（ランチャー）: `GAMES` 配列に1行足すだけのデータ駆動

## これまでの学び
- **単一ファイル静的構成**は、配布・ホスティング・オフライン性で5歳向け小品に最適。
- **演出（音・揺れ・ほめ言葉）が「気持ちよさ」と継続意欲に直結**。地味な正解より派手な達成感。
- レスポンシブは**横向き（高さ制約）が要注意**。盤は `min(幅, 高さ)` で決め、回転時は状態を保ったまま組み直す。
- **CSSの `@media` 上書きはソース順**。基本ルールより後ろ（末尾）に置く。
- テストは**ローカルHTTP（`file://` 不可）＋複数画面サイズの実測**が有効。はみ出しと操作可否を数値で確認。
- 公開は **GitHub Pages が最短**。push するだけで反映。`.claude/` は gitignore で必ず除外。

## ロードマップ / 今後のアイデア
**新ゲーム候補**
- すうじ・かずあそび、神経衰弱（memory-card）、いろあわせ、めいろ、おえかき、おとあそび、とけいのよみかた

**共通化**
- `shared/` に共通CSS（パレット/ボタン/猫）とヘルパJS（Web Audio の `beep`、猫リアクション `catReact`、`computeCellPx`/`relayout`）を切り出し、各ゲームから読み込む

**メニュー / 体験**
- 「じゅんびちゅう」カード、最近あそんだゲーム、★獲得数の保存（localStorage）
- 音量トグル、文字サイズ、配色コントラストなどアクセシビリティ

**基盤**
- PWA化（manifest + service worker）でホーム追加・オフライン起動
- favicon / アプリアイコン整備、metaタグ更新（`mobile-web-app-capable`）
